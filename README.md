# 대한항공 Clone Project

HTML, CSS를 이용하여 스터디를 진행하는 Team 프로젝트로 멤버들이 대한항공 사이트 내의 페이지를 선택하여 1페이지씩 클론하였습니다.

## 프로젝트 진행 방법 및 목표

- 페이지 개별 작업 진행 및 제출
- HTML, CSS 익히기
- 시멘틱 마크업
- 스쿨 과정 종료 후 vanilla JS 사용하여 기능 개발 진행
